Project Description

Group project for a university Deep Learning course, developing and evaluating convolutional neural networks (CNNs) for image classification tasks.

Skills & Technologies

  • Python, PyTorch, and Jupyter Notebook
  • Model design: CNN architectures, layers, activation functions

  • Hyperparameter tuning and model evaluation
  • Team collaboration and reproducible research

Code Logic

  • Preprocesses image datasets and applies augmentation for training/validation splits.
  • Defines CNN architectures using PyTorch modules (Conv2d, ReLU, MaxPool, etc.).
  • Trains models using backpropagation, tracks loss/accuracy, and prevents overfitting (dropout, early stopping).
  • Jupyter Notebooks integrate code, results, and visualizations for easy review.

Sample Code

# Example: Simple CNN in PyTorch
import torch.nn as nn
class SimpleCNN(nn.Module):
    def __init__(self):
        super().__init__()
        self.conv = nn.Conv2d(3, 32, 3, padding=1)
        self.pool = nn.MaxPool2d(2, 2)
        self.fc = nn.Linear(32*16*16, 10)
    def forward(self, x):
        x = self.pool(nn.functional.relu(self.conv(x)))
        x = x.view(x.size(0), -1)
        return self.fc(x)
