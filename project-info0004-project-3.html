Project Description

INFO0004 - Object-oriented programming project 3: a software engineering exercise focused on implementing advanced OOP concepts in C++.

Skills & Technologies

  • C++ programming (classes, inheritance, polymorphism)
  • Design Patterns (e.g., Factory, Observer)
  • Unit testing and debugging
  • Makefile automation
  • Code documentation and UML diagrams

Code Logic

  • Defines a set of base and derived classes to model entities and behaviors.
  • Implements design patterns for extensibility and maintainability.
  • Encapsulates data and logic using C++ best practices.
  • Automates builds and testing via Makefile targets.
  • Includes test cases for all major class interactions.
