Distributed Computing: MapReduce & Spark

Python Spark MapReduce

Project Description

University course project focused on distributed data processing using MapReduce and Apache Spark. The goal was to understand and implement scalable algorithms for big data analytics.

Skills & Technologies

  • Python for scripting and algorithm development
  • Apache Spark for distributed computation
  • MapReduce paradigm for scalable data processing
  • Cluster computing concepts
  • Data analysis and reporting

Code Logic

  • Implemented classic MapReduce algorithms (e.g., word count, join, aggregation) in Python.
  • Used PySpark to process large datasets in parallel across a cluster.
  • Optimized data pipelines for performance and scalability.
  • Analyzed and visualized results to report on efficiency and accuracy.

Sample Code

# Example: Word count with PySpark
from pyspark import SparkContext
sc = SparkContext()
counts = sc.textFile("data.txt") \
           .flatMap(lambda line: line.split()) \
           .map(lambda word: (word, 1)) \
           .reduceByKey(lambda a, b: a + b)
print(counts.collect())
