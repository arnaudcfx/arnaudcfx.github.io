Project Description

The second collaborative project in Operating Systems, extending our OS kernel to include advanced process and memory management features.

Skills & Technologies

  • C programming (systems-level, memory management)
  • Process management and scheduling policies
  • Memory paging and virtual address space
  • Debugging concurrent systems
  • Version control with Git

Code Logic

  • Implemented process table and new scheduling strategies (priority, round robin).
  • Added support for virtual memory and page tables.
  • Designed interrupt handlers for timer and memory faults.
  • Integrated extensive logging and debugging macros.
  • Tested edge cases such as process starvation and memory leaks.

Sample Code

// Page fault handler (simplified)
void handle_page_fault(proc_t *p, uint32_t addr) {
    if (not_mapped(addr)) {
        map_page(p, addr);
    } else {
        kill_process(p);
    }
}

Results & Impact

  • Successfully implemented virtual memory and preemptive multitasking.
