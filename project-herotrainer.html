HeroTrainer

Makefile Java Dart HTML

Project Description

HeroTrainer is a cross-platform training application combining Java, Dart, and HTML for both backend and frontend logic. It is designed as an educational tool or game where users can train virtual heroes, complete tasks, and track progress.

Skills & Technologies

  • Java for core logic and backend services
  • Dart for Flutter-based UI and mobile/web integration
  • HTML for frontend content display
  • Makefile for build automation
  • OOP, event-driven programming, and cross-language integration

Code Logic

  • The backend in Java handles user data, hero stats, and game logic.
  • Dart/Flutter provides a responsive, interactive UI for users to interact with their heroes and view stats in real time.
  • HTML is used to render help pages or static content within the app.
  • Makefile scripts automate building, testing, and deployment.

Sample Code

// Example: Hero class in Java
public class Hero {
    private String name;

Results & Impact

  • Successfully integrated Java backend with Flutter UI for a smooth user experience.
  • Developed reusable modules for hero management and event handling.
  • Learned best practices in multi-language project structure and automated builds.
