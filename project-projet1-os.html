Projet1-OS

C Makefile

Project Description

The first group project in Operating Systems, focusing on foundational OS concepts and the implementation of basic process management and scheduling.

Skills & Technologies

  • C programming (systems-level)
  • Process management, context switching
  • Makefile build automation
  • Unit and integration testing of system modules
  • Git for collaborative development

Code Logic

  • Implements process creation, switching, and termination using C structs and manual memory management.
  • Scheduler selects the next process to run (e.g., round-robin or priority-based).
  • Interrupt handlers manage time slices and context saving/restoring.
  • Testing framework verifies correct process transitions and system stability.

Sample Code

// Process context switch (simplified)
void context_switch(process_t *from, process_t *to) {
    save_context(from);
    restore_context(to);
    current_process = to;
}

Results & Impact

  • Demonstrated correct process switching and scheduling under simulated OS conditions.
  • Established a strong foundation for more advanced OS projects.
  • Improved team collaboration and code review practices.
