Project Description

INFO-2055 Embedded Systems project: a microcontroller-based customer counter for real-time monitoring of entries in a store or room. Designed for reliability, accuracy, and ease of integration in physical environments.

Skills & Technologies

  • Embedded Assembly programming (targeting AVR microcontrollers)
  • Real-time interrupt handling
  • Digital sensor interfacing (IR, pushbuttons)
  • Makefile automation for firmware build & flash
  • Hardware debugging and oscilloscope use

Code Logic

  • On each sensor activation (entry/exit), an interrupt service routine (ISR) is triggered.
  • The ISR debounces the input, increments or decrements a memory-stored counter, and updates the display.
  • All I/O (buttons, display) is handled via direct memory-mapped register manipulation.
  • A main loop keeps the system in low-power until an interrupt occurs.
  • Build automation is handled by a Makefile and code is modularized for easy testing.

Sample Code

; Interrupt Service Routine for Button Press
ISR(INT0_vect)
    ; Debounce input and increment counter
    lds   r16, customer_count
    inc   r16
    sts   customer_count, r16
    ; Update display routine
    rcall update_display
    reti

Results & Impact

  • Accurately counted customer entries/exits in real-time with zero missed events in tests.
  • Tested in a lab environment; validated for robustness against electrical noise/glitches.
  • Gained deep experience in real-time embedded system design and debugging.
