Customer Counter

Individual project
Python Flask OpenCV JavaScript

Summary

Developed a web app to count customers entering a store using a camera feed and display real-time stats on a dashboard. Used Flask for backend, OpenCV for video processing, and JavaScript for the frontend dashboard.

Skills & Technologies

  • Real-time video processing with OpenCV
  • Web backend with Flask
  • Interactive JavaScript dashboard
  • Deployment with Docker

Sample Code


import cv2

cap = cv2.VideoCapture(0)
while True:
    ret, frame = cap.read()
    # ... customer detection logic ...
    cv2.imshow('frame', frame)
    if cv2.waitKey(1) & 0xFF == ord('q'):
        break
cap.release()
cv2.destroyAllWindows()

Results

  • Live demo deployed for a local shop
  • Dashboard shows real-time and historical customer counts
