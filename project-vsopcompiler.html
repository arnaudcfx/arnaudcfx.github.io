Project Description

A full compiler for the VSOP language, created as a systems programming project. The compiler translates VSOP source code to LLVM bytecode, supporting custom language features and optimizations.

Skills & Technologies

  • C++ for compiler modules (parsing, AST, codegen)
  • Lex and Yacc for lexical/syntax analysis
  • LLVM for intermediate code generation and optimization
  • Makefile for build automation
  • Unit testing, debugging, and CI pipelines

Code Logic

  • Lex and Yacc define the lexical and syntactic structure of the VSOP language.
  • C++ modules build an abstract syntax tree (AST) and perform semantic analysis (type checking, etc.).
  • Generates LLVM IR, which is then compiled to target machine code.
  • Implements error handling, reporting, and code optimizations.
