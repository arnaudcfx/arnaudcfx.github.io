INFO0009-Site

PHP Hack

Project Description

Developed as part of the INFO0009 course, this project is a dynamic web platform built with PHP and Hack. It features data entry forms, user authentication, and information display, giving students a hands-on experience with server-side web development.

Skills & Technologies

  • PHP and Hack for backend scripting and templating
  • HTML/CSS for frontend structure and style
  • Session management and user authentication
  • Form validation and secure input handling
  • Database interactions (CRUD operations)

Code Logic

  • Implements a login system storing user credentials securely using PHP sessions.
  • Processes and validates user forms, providing feedback for errors or successful submissions.
  • Generates dynamic web pages by querying and updating a database using PHP/Hack.
  • Implements modular code to separate logic, data, and presentation layers for maintainability.

Sample Code

<?php
// User login handler (simplified)
session_start();
if ($_SERVER['REQUEST_METHOD'] === 'POST') {
    $user = $_POST['user'];
    $pass = $_POST['pass'];
    // Validate credentials (pseudo-code)
    if (check_credentials($user, $pass)) {
        $_SESSION['user'] = $user;
        header('Location: dashboard.php');
    } else {
        echo "Invalid login!";
    }
}
?>

Results & Impact

  • Enabled students to practice real-world web development with security and usability in mind.
  • Successfully demonstrated CRUD, authentication, and session management.
  • Provided a base for further learning in web technologies and backend frameworks.
