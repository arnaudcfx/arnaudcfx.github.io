MathAlgo

C Makefile

Project Description

Implementation of sparse matrix operations for scientific computing. Focused on efficient storage and manipulation of large, mostly-zero matrices.

Skills & Technologies

  • Data structures for sparse matrix representation
  • Efficient matrix arithmetic (add, multiply, transpose)
  • Memory management in C
  • Command-line interface and file I/O
  • Testing and validation with numerical datasets

Code Logic

  • Custom structs represent matrices by only storing non-zero entries (row, column, value).
  • Arithmetic functions traverse sparse representations for addition/multiplication.
  • Efficient memory allocation and deallocation to avoid leaks.
  • CLI allows loading, saving, and operating on matrices from files or user input.
