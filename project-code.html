Introduction to Stochastic Processes: Discrete-Time Markov Chains

Project Summary

This project explores the SIR epidemic model using discrete-time Markov chains. The goal is to simulate and analyze the spread of an infectious disease within a population, studying the transitions between three key states: Susceptible (S), Infected (I), and Recovered (R).

The report demonstrates how the evolution of the system depends only on its current state (Markov property) and how probabilistic transitions can model infection and recovery dynamics. Using adjacency matrices, the propagation of infection is observed under different configurations — linear (Wlin), fully connected (Wfull), and large networks (Wbig).

Multiple simulations were run to compare these network structures and to evaluate the impact of various mitigation strategies (infection reduction, confinement, medication, vaccination) on epidemic duration and hospital capacity.

Key Findings

  • The SIR model follows the Markov property: future states depend only on the current state.
  • Networks with more connections (e.g., Wfull) produce faster and more severe epidemics.
  • The average epidemic duration varied between 11 and 47 days depending on network structure.
  • Intervention measures (reduced infection probability, vaccination, etc.) significantly lower the infection peak and hospital overload.
  • Simulated results converge to theoretical values after approximately 500 runs.

Technologies & Tools Used

  • Python – for modeling, data simulation, and numerical computation.
  • NumPy / Matplotlib – to manage matrices and visualize epidemic dynamics.
  • Markov chain theory – for transition matrices and steady-state analysis.
  • LaTeX / HTML – for documentation and report formatting.
  • Statistical analysis – to interpret simulation outcomes and compare models.
