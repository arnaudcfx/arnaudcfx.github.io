Project Description

A collection of Python scripts and utilities for automation, data processing, file handling, and rapid prototyping.

Skills & Technologies

  • Data processing, API requests, and file I/O
  • CLI argument parsing (argparse, sys.argv)

  • Basic error handling and logging

Code Logic

  • Each script is modular, with a clear entry point and reusable functions.
  • Common patterns: reading/writing files, parsing data, making API requests, or automating tasks.
  • Many scripts use argument parsing for flexibility and automation.
  • Includes utility functions for data cleaning and transformation.

Sample Code

# Example: Batch rename files in a folder
import os
for i, fname in enumerate(os.listdir("data")):
    os.rename(f"data/{fname}", f"data/file_{i}.txt")
