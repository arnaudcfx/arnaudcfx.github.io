C Jupyter Notebook Shell

Project Description

University group project focused on designing and implementing highly efficient algorithms for scientific data analysis and simulation, leveraging parallel computing techniques.

Skills & Technologies

  • Parallel programming in C (OpenMP, MPI)
  • Scientific computing, numerical analysis

Code Logic

  • Numerical algorithms (e.g., matrix multiplication, FFT, PDE solvers) coded in C.
  • Parallelized using OpenMP/MPI: distributes computation across CPU cores or nodes.
  • Performance bottlenecks are analyzed and code is iteratively optimized.
The full source code is not published to respect the academic and group work confidentiality.
