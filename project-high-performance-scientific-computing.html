Project Description

University group project focused on designing and implementing highly efficient algorithms for scientific data analysis and simulation, leveraging parallel computing techniques.

Skills & Technologies

  • Parallel programming in C (OpenMP, MPI)
  • Scientific computing, numerical analysis
  • Jupyter Notebook for data visualization and results
  • Shell scripting for automation and reproducibility
  • Profiling, benchmarking, and optimization

Code Logic

  • Numerical algorithms (e.g., matrix multiplication, FFT, PDE solvers) coded in C.
  • Parallelized using OpenMP/MPI: distributes computation across CPU cores or nodes.
  • Jupyter Notebooks visualize results and compare speedups between serial/parallel code.
  • Shell scripts automate compilation, running, and result collection for benchmarks.
  • Performance bottlenecks are analyzed and code is iteratively optimized.

Sample Code

// Parallel matrix multiplication with OpenMP
#pragma omp parallel for shared(A, B, C)
for (int i = 0; i < N; ++i)
  for (int j = 0; j < N; ++j) {
    C[i][j] = 0;
    for (int k = 0; k < N; ++k)
      C[i][j] += A[i][k] * B[k][j];
  }

Results & Impact

  • Achieved up to 8x speedup on multicore hardware versus serial code.
  • Produced detailed reports and visualizations of algorithm scaling.
  • Developed strong practical skills in parallel programming and scientific research.
