C Makefile

Project Description

The third and final group project in Operating Systems: implementing a mini operating system with full process, memory, and I/O management.

Skills & Technologies

  • C programming (kernel-level, concurrency)
  • Process and memory management
  • File system and device driver basics
  • Test-driven development
  • Technical documentation and teamwork

Code Logic

  • Integrated all previous modules into a single, multi-tasking OS kernel.
  • Implemented scheduling, paging, file I/O, and device management.
  • Developed user programs to stress-test the OS.
  • Added logging and runtime error handling for system stability.
  • Collaborated on documentation and code reviews.

Sample Code

// Scheduler main loop (simplified)
while (1) {
    proc_t *p = next_ready_process();
    if (p) run_process(p);
    else idle();
}

Results & Impact

  • Delivered a working, multi-process OS prototype with basic file and device support.
  • Enhanced skills in systems programming and group project management.
  • Prepared team for more advanced OS and hardware projects.
